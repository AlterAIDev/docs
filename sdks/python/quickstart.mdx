## Install

```bash
pip install alter-sdk
```

Requires Python 3.10+.

## Instantiate

```python
from alter_sdk import App

app = App(api_key="alter_key_app_...")
```

For agents (workload identity), use `Agent` instead:

```python
from alter_sdk import Agent

agent = Agent(api_key="alter_key_agent_...")
```

See [Agents → Overview](/agents/overview) for when each class applies.

## Make a call

```python
import asyncio
from alter_sdk import App, HttpMethod

async def main():
    app = App(api_key="alter_key_app_...")
    response = await app.request(
        HttpMethod.GET,
        "https://www.googleapis.com/calendar/v3/calendars/primary/events",
        grant_id="grant_...",
    )
    print(response.json())
    await app.close()

asyncio.run(main())
```

The SDK is fully async. For sync code, run inside `asyncio.run(...)`.

## Identity resolution modes

Three options. Pick per app, or per call:

<CodeGroup>
```python JWT (recommended)
app = App(
    api_key="alter_key_app_...",
    user_token_getter=lambda: get_jwt_from_request(),
)
await app.request("GET", url, provider="google")
```

```python grant_id
app = App(api_key="alter_key_app_...")
await app.request("GET", url, grant_id="grant_...")
```

```python Headless
app = App(api_key="alter_key_app_...")
results = await app.connect(providers=["github"])
await app.request("GET", url, grant_id=results[0].grant_id)
```
</CodeGroup>

See [Concepts → Identity resolution](/concepts/identity-resolution).
