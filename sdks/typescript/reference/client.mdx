## `App`

The operator/admin client. Constructed with an app key (`alter_key_app_…`).

### Signature

```typescript
new App(options: {
  apiKey: string;
  timeout?: number;
  caller?: string;
  callerType?: CallerType;
  userTokenGetter?: () => string | Promise<string>;
});
```

### Parameters

| Name | Type | Default | Description |
|---|---|---|---|
| `apiKey` | `string` | required | App API key. |
| `timeout` | `number` | `30000` | HTTP request timeout in ms. |
| `caller` | `string` | `undefined` | Audit attribution label. Free-form string or a managed-agent UUID. See [Concepts → Caller](/concepts/principals-and-grants#caller). |
| `callerType` | `CallerType` | `SERVICE` | `SERVICE` (default) or `AGENT`. |
| `userTokenGetter` | function | `undefined` | Sync or async function returning a JWT for [identity resolution](/concepts/identity-resolution). |

### Properties

| Property | Type | Notes |
|---|---|---|
| `actorId` | `string \| null` | Cached actor UUID. |
| `agents` | `AgentsNamespace` | See [Agents reference](/sdks/typescript/reference/agents). |
| `baseUrl` | `string` | Resolved Alter endpoint. Diagnostic only. |
| `lastRetryInfo` | `RetryInfo \| null` | Retry metadata from the most recent `request()`. |

### Example

```typescript
import { App } from "@alter-ai/alter-sdk";

const app = new App({
  apiKey: "alter_key_app_...",
  caller: "email-research-bot",
  userTokenGetter: () => getJwt(),
});
```

## `Agent`

The workload client. Constructed with an agent key (`alter_key_agent_…`).

### Signature

```typescript
new Agent(options: {
  apiKey: string;
  timeout?: number;
  caller?: string;
  userTokenGetter?: () => string | Promise<string>;
});
```

### Parameters

Same as `App`, except `callerType` is implicit (always `AGENT`). `userTokenGetter` is allowed but **per-call combination of an agent key with a `userToken` returns HTTP 400** — see [Agents → Overview](/agents/overview).

### Example

```typescript
import { Agent } from "@alter-ai/alter-sdk";

const agent = new Agent({ apiKey: "alter_key_agent_..." });
const me = await agent.me();
console.log(me.id, me.name);
```

## Lifecycle

Always call `close()` to release the underlying HTTP pool when the SDK instance is no longer needed:

```typescript
const app = new App({ apiKey: KEY });
try {
  const response = await app.request("GET", url, { grantId: g });
} finally {
  await app.close();
}
```
