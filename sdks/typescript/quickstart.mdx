## Install

```bash
npm install @alter-ai/alter-sdk
```

Works with Node.js 18+ and any modern bundler.

## Instantiate

```typescript
import { App } from "@alter-ai/alter-sdk";

const app = new App({ apiKey: "alter_key_app_..." });
```

For agents (workload identity), use `Agent` instead:

```typescript
import { Agent } from "@alter-ai/alter-sdk";

const agent = new Agent({ apiKey: "alter_key_agent_..." });
```

See [Agents → Overview](/agents/overview) for which class to use when.

## Make a call

```typescript
import { App, HttpMethod } from "@alter-ai/alter-sdk";

const app = new App({ apiKey: "alter_key_app_..." });
try {
  const response = await app.request(
    HttpMethod.GET,
    "https://www.googleapis.com/calendar/v3/calendars/primary/events",
    { grantId: "grant_..." },
  );
  console.log(await response.json());
} finally {
  await app.close();
}
```

The SDK is promise-based. Use `await` everywhere.

## Identity resolution modes

Three options. Pick per app, or per call:

<CodeGroup>
```typescript JWT (recommended)
const app = new App({
  apiKey: "alter_key_app_...",
  userTokenGetter: () => getJwtFromRequest(),
});
await app.request("GET", url, { provider: "google" });
```

```typescript grant_id
const app = new App({ apiKey: "alter_key_app_..." });
await app.request("GET", url, { grantId: "grant_..." });
```

```typescript Headless
const app = new App({ apiKey: "alter_key_app_..." });
const results = await app.connect({ providers: ["github"] });
await app.request("GET", url, { grantId: results[0].grantId });
```
</CodeGroup>

See [Concepts → Identity resolution](/concepts/identity-resolution).

## User → agent delegation for managed secrets

When an end user authorizes an agent to USE the user's existing managed-secret access, the flow goes through a Connect approval screen. The SDK mints the session; the user consents in a browser; the resulting delegation id is passed back as a regular `grantId` to the agent's `request()`.

```typescript
const billingBotInfo = await app.agents.getByName("billing-bot");
const session = await app.createManagedSecretConnectSession({
  templateSlug: "stripe-api-key",
  delegatedAgentId: billingBotInfo.id,
  userToken: userJwt,
});
// Send the user to session.connectUrl.
```

See [Managed Secrets → Delegation](/managed-secrets/delegation) for the full flow + runtime semantics.
