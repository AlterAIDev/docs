## Why Choose Alter Vault?

### Never Store Tokens
Your database stays clean. We handle all token storage, encryption, and management in our secure infrastructure.

### Automatic Token Refresh
Tokens are refreshed automatically before they expire. Your API calls never fail due to expired credentials.

### Enterprise Security
Bank-grade encryption, SOC 2 compliance, and complete audit trails protect your users' data.

### Built for AI Agents
First-class actor tracking for AI agents and MCP servers. Know exactly which agent accessed what data and when, with per-request context (run ID, thread ID, tool call ID).

### Complete Audit Trail
Every token access, connection change, and API call is logged automatically. View, filter, and export logs from the dashboard for SOC 2, GDPR, and HIPAA compliance.

### Universal Compatibility
Works with your existing authentication system. Use JWT tokens from Auth0, Clerk, Supabase, or any provider.

## How It Works

<Steps>
  <Step title="Connect or Store Credentials">
    **OAuth**: User authorizes via Alter Connect UI. **Managed Secrets**: Store your API keys in the Developer Portal.
  </Step>

  <Step title="We Encrypt & Store">
    All credentials are encrypted (AES-256-GCM) and stored in our secure vault — never in your database
  </Step>

  <Step title="You Make API Calls">
    Call `vault.request()` with a `connection_id` and we inject the right credential automatically
  </Step>

  <Step title="We Handle the Rest">
    Token refresh, policy enforcement, audit logging — all automatic
  </Step>
</Steps>

## Quick Example

<CodeGroup>

```python Python (Headless)
import asyncio
from alter_sdk import AlterVault, ActorType, HttpMethod

async def main():
    vault = AlterVault(
        api_key="alter_key_...",
        actor_type=ActorType.AI_AGENT,
        actor_identifier="my-agent",
    )

    # Open browser → user authorizes → SDK returns connection
    result = await vault.connect(
        providers=["google"],
    )

    # Make API calls with the connection
    response = await vault.request(
        result.connection_id,
        HttpMethod.GET,
        "https://www.googleapis.com/calendar/v3/calendars/primary/events",
        query_params={"maxResults": "10"},
    )
    events = response.json()
    print(f"Found {len(events.get('items', []))} events")

    await vault.close()

asyncio.run(main())
```

```javascript JavaScript (Web App)
import AlterConnect from '@alter-ai/connect';

// Create Connect instance
const alterConnect = AlterConnect.create({
    customization: {
        colors: { primary: '#6366f1' }
    }
});

// Open OAuth flow
async function connectProvider() {
    // Call YOUR backend to create a session
    const { session_token } = await fetch('/api/connect/session')
        .then(r => r.json());

    await alterConnect.open({
        token: session_token,
        onSuccess: (connection) => {
            console.log('Connected!', connection);
        }
    });
}
```

```python Backend (Session Creation)
from alter_sdk import AlterVault, ActorType

vault = AlterVault(
    api_key="alter_key_...",
    actor_type=ActorType.BACKEND_SERVICE,
    actor_identifier="my-app",
)

# Create session for frontend
session = await vault.create_connect_session()
session_token = session.session_token
```
