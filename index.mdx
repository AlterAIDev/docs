---
title: 'Alter'
description: 'Scoped credentials for apps and AI agents — without storing tokens, building OAuth flows, or running an audit pipeline.'
---

Alter sits between application code and third-party APIs. Application code calls one method; Alter injects the credential, refreshes it, enforces policy, and writes the audit row.
