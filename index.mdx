## Why Choose Alter Vault?

### Never Store Tokens
Your database stays clean. We handle all token storage, encryption, and management in our secure infrastructure.

### Automatic Token Refresh
Tokens are refreshed automatically before they expire. Your API calls never fail due to expired credentials.

### Enterprise Security
Bank-grade encryption, SOC 2 compliance, and complete audit trails protect your users' data.

### Universal Compatibility
Works with your existing authentication system. Use JWT tokens from Auth0, Clerk, Supabase, or any provider.

## Quick Example

<CodeGroup>

```python Python
import asyncio
from alter_sdk import AlterVault, Provider

async def use_google_calendar():
    # Initialize the SDK
    vault = AlterVault(
        api_key="alter_key_...",
        app_id="app_..."
    )

    # Get Google client - tokens injected automatically!
    google = await vault.get_client(
        provider=Provider.GOOGLE,
        user={"user_id": "alice"}
    )

    # Use Google APIs normally - no token handling!
    calendar = await google.build("calendar", "v3")
    events = calendar.events().list(
        calendarId="primary"
    ).execute()

    await vault.close()

asyncio.run(use_google_calendar())
```

```javascript JavaScript
import AlterConnect from '@alter-ai/connect';

// Create Connect instance
const alterConnect = AlterConnect.create({
    customization: {
        colors: { primary: '#6366f1' }
    }
});

// Open OAuth flow
async function connectProvider() {
    // Get session from YOUR backend
    const { session_token } = await fetch('/api/oauth/session')
        .then(r => r.json());

    await alterConnect.open({
        token: session_token,
        onSuccess: (connection) => {
            console.log('Connected!', connection);
        }
    });
}
```

```python Direct API
import requests

# Create session for frontend
response = requests.post(
    "https://api.alter.com/oauth/connect/session",
    headers={"x-api-key": "alter_key_..."},
    json={
        "end_user": {"id": "user_123"},
        "attributes": {"user_id": "user_123"}
    }
)
session_token = response.json()["session_token"]

# Get token for backend use
response = requests.post(
    "https://api.alter.com/oauth/token",
    headers={"x-api-key": "alter_key_..."},
    json={
        "provider_id": "google",
        "attributes": {"user_id": "user_123"}
    }
)
access_token = response.json()["access_token"]
```
