---
title: 'Managed secret providers'
description: 'Operator-owned API keys for backend services across supported providers.'
---

Alter ships with 62 pre-configured managed-secret templates covering AI tooling, REST-native databases and search engines, analytics, communication, payments, infrastructure, error tracking, and developer tools. Each provider has a setup guide with the property table, credential creation flow, and SDK code example.

For the end-to-end flow, see [Provision secrets for backend services](/guides/provision-secrets-for-backend-services). For the conceptual model, see [Managed secrets](/concepts/credentials#managed-secrets).

For any header-based credential not in this catalog, use the **Custom** template — configure header name, injection format (`Bearer {token}`, `Token {token}`, `Key {token}`, …), and multi-header or query-parameter injection as needed.
