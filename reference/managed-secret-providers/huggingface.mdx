## Overview

Use Hugging Face credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `huggingface` |
| **Category** | AI |
| **Credential Type** | Access Token |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to Hugging Face">
    Log in to Hugging Face at [huggingface.co](https://huggingface.co/).
  </Step>
  <Step title="Navigate to Access Tokens">
    Navigate to Settings > Access Tokens at [huggingface.co/settings/tokens](https://huggingface.co/settings/tokens).
  </Step>
  <Step title="Create a token">
    Click **Create new token** and choose a role (fine-grained, read, or write).
  </Step>
  <Step title="Copy the token">
    Copy the token (starts with `hf_`) — it is only shown once.
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Hugging Face">
    Go to **Managed Secrets** > **Add Provider** > **Hugging Face**.
  </Step>

  <Step title="Enter credentials">
    Paste your Access Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.GET,
    "https://huggingface.co/api/whoami-v2",
    grant_id="YOUR_GRANT_ID",
)
```
