## Overview

Use Milvus credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `milvus` |
| **Category** | AI |
| **Credential Type** | API Key |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to Zilliz Cloud">
    Log in to [Zilliz Cloud](https://cloud.zilliz.com/) (hosted Milvus).
  </Step>
  <Step title="Create an API key">
    Open **API Keys** and create a key for the organization or project.
  </Step>
  <Step title="Copy the key">
    Copy the key — it is only shown once.
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Milvus">
    Go to **Managed Secrets** > **Add Provider** > **Milvus**.
  </Step>

  <Step title="Enter credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.POST,
    "https://YOUR-CLUSTER.cloud.zilliz.com/v2/vectordb/entities/search",
    grant_id="YOUR_GRANT_ID",
    json={
        "collectionName": "my_collection",
        "data": [[0.05, 0.61, 0.76, 0.74]],
        "limit": 5,
    },
)
```

## Notes

- Self-hosted Milvus accepts `username:password` as the bearer value on the RESTful API.
