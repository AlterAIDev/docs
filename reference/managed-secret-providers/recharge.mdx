## Overview

Use Recharge credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `recharge` |
| **Category** | E-Commerce |
| **Credential Type** | API Key |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to the Recharge Merchant Portal">
    Log in to the Recharge Merchant Portal — accessed from the ecommerce platform admin (e.g. Shopify/BigCommerce → Apps → Recharge) or the Recharge direct login page.
  </Step>
  <Step title="Go to API tokens">
    Go to Tools & apps → API tokens. See the [Recharge API token guide](https://docs.getrecharge.com/docs/recharge-api-key) for details.
  </Step>
  <Step title="Create an API token">
    Click 'Create an API token' and configure permissions.
  </Step>
  <Step title="Copy your token">
    Copy the generated API token.
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Recharge">
    Go to **Managed Secrets** > **Add Provider** > **Recharge**.
  </Step>

  <Step title="Enter credentials">
    Paste your API Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.GET,
    "https://api.rechargeapps.com/subscriptions",
    grant_id="YOUR_GRANT_ID",
)
```
