## Overview

Use Replicate credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `replicate` |
| **Category** | AI |
| **Credential Type** | API Token |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to Replicate">
    Log in to Replicate at [replicate.com](https://replicate.com/).
  </Step>
  <Step title="Navigate to API Tokens">
    Navigate to Account > API Tokens at [replicate.com/account/api-tokens](https://replicate.com/account/api-tokens).
  </Step>
  <Step title="Create or copy a token">
    Create a new token or copy the default one (starts with `r8_`).
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Replicate">
    Go to **Managed Secrets** > **Add Provider** > **Replicate**.
  </Step>

  <Step title="Enter credentials">
    Paste your API Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.POST,
    "https://api.replicate.com/v1/predictions",
    grant_id="YOUR_GRANT_ID",
)
```
