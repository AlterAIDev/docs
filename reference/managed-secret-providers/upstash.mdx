## Overview

Use Upstash credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `upstash` |
| **Category** | Database |
| **Credential Type** | REST Token |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to the Upstash Console">
    Log in to the [Upstash Console](https://console.upstash.com/).
  </Step>
  <Step title="Open the REST API section">
    Open the database and find the **REST API** section.
  </Step>
  <Step title="Copy the token">
    Copy the `UPSTASH_REDIS_REST_TOKEN` (or the read-only token for read-only integrations).
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Upstash">
    Go to **Managed Secrets** > **Add Provider** > **Upstash**.
  </Step>

  <Step title="Enter credentials">
    Paste your REST Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.GET,
    "https://YOUR-DATABASE.upstash.io/get/mykey",
    grant_id="YOUR_GRANT_ID",
)
```

## Notes

- Each database has its own REST URL and token pair.
