## Overview

Use AssemblyAI credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `assemblyai` |
| **Category** | AI |
| **Credential Type** | API Key |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to the AssemblyAI Dashboard">
    Log in to the AssemblyAI Dashboard at [assemblyai.com/app](https://www.assemblyai.com/app/).
  </Step>
  <Step title="Navigate to API Keys">
    Navigate to API Keys at [assemblyai.com/app/api-keys](https://www.assemblyai.com/app/api-keys).
  </Step>
  <Step title="Copy a key">
    Copy the default key or create a new one.
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add AssemblyAI">
    Go to **Managed Secrets** > **Add Provider** > **AssemblyAI**.
  </Step>

  <Step title="Enter credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.POST,
    "https://api.assemblyai.com/v2/transcript",
    grant_id="YOUR_GRANT_ID",
)
```

## Notes

- The bare key is injected into the `Authorization` header (no `Bearer` prefix).
