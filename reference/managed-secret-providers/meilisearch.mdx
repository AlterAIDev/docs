## Overview

Use Meilisearch credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `meilisearch` |
| **Category** | Search |
| **Credential Type** | API Key |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to Meilisearch Cloud">
    Log in to [Meilisearch Cloud](https://cloud.meilisearch.com/) and open the project.
  </Step>
  <Step title="Open API keys">
    Open **Settings** > **API keys**.
  </Step>
  <Step title="Copy a key">
    Copy the Default Search API Key (read) or Default Admin API Key (read/write).
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Meilisearch">
    Go to **Managed Secrets** > **Add Provider** > **Meilisearch**.
  </Step>

  <Step title="Enter credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.POST,
    "https://YOUR-PROJECT.meilisearch.io/indexes/movies/search",
    grant_id="YOUR_GRANT_ID",
    json={"q": "interstellar", "limit": 5},
)
```
