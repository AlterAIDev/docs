## Overview

Use Firecrawl credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `firecrawl` |
| **Category** | Search |
| **Credential Type** | API Key |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to the Firecrawl Dashboard">
    Log in to the Firecrawl Dashboard at [firecrawl.dev/app](https://www.firecrawl.dev/app).
  </Step>
  <Step title="Navigate to API Keys">
    Navigate to API Keys at [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys).
  </Step>
  <Step title="Create or copy a key">
    Create a new API key or copy the default one (starts with `fc-`).
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Firecrawl">
    Go to **Managed Secrets** > **Add Provider** > **Firecrawl**.
  </Step>

  <Step title="Enter credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.POST,
    "https://api.firecrawl.dev/v2/scrape",
    grant_id="YOUR_GRANT_ID",
)
```
