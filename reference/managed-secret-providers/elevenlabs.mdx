## Overview

Use ElevenLabs credentials to make authenticated API calls through Alter Vault without exposing API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `elevenlabs` |
| **Category** | AI |
| **Credential Type** | API Key |

## Step 1: Get Credentials

<Steps>
  <Step title="Log in to ElevenLabs">
    Log in to ElevenLabs at [elevenlabs.io](https://elevenlabs.io/app/).
  </Step>
  <Step title="Navigate to API Keys">
    Navigate to Settings > API Keys at [elevenlabs.io/app/settings/api-keys](https://elevenlabs.io/app/settings/api-keys).
  </Step>
  <Step title="Create a key">
    Click **Create API Key**, name it, and optionally restrict permissions.
  </Step>
  <Step title="Copy the key">
    Copy the key — it is only shown once.
  </Step>
</Steps>

<Warning>
  Never share credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add ElevenLabs">
    Go to **Managed Secrets** > **Add Provider** > **ElevenLabs**.
  </Step>

  <Step title="Enter credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await alter_app.request(
    HttpMethod.GET,
    "https://api.elevenlabs.io/v1/voices",
    grant_id="YOUR_GRANT_ID",
)
```

## Notes

- The key is injected via the `xi-api-key` header.
