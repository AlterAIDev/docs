The TypeScript SDK ships as the [`@alter-ai/alter-sdk` package on npm](https://www.npmjs.com/package/@alter-ai/alter-sdk). Node 18+.

This is the **server-side** SDK (Node, Bun, Deno, edge runtimes with `fetch`). It is not for browser code — for the browser-side OAuth widget that opens the consent popup, see the [Connect SDK](/reference/connect-sdk).

```bash
npm install @alter-ai/alter-sdk
```

```typescript
import { App, Agent, HttpMethod } from "@alter-ai/alter-sdk";
```

## Two clients

The SDK exposes two top-level clients. They share the request surface but differ in principal and identity scoping.

<CardGroup cols={2}>
  <Card title="App" icon="building" href="/reference/typescript-sdk/client#app">
    Construct with an `alter_key_app_…` key. Acts as the application — provisions agents, mints Connect sessions, manages grants, and proxies provider calls under stored grants.
  </Card>
  <Card title="Agent" icon="robot" href="/reference/typescript-sdk/client#agent">
    Construct with an `alter_key_agent_…` key. Workload-scoped: can only reach credentials delegated or directly bound to the agent. The right entry point for AI agents and tool-calling workloads.
  </Card>
</CardGroup>

Both classes are async. Always close them (`await app.close()`) or use `await using` for automatic disposal (TypeScript 5.2+, Node 18+).

```typescript
await using app = new App({ apiKey: process.env.ALTER_API_KEY! });
const page = await app.listGrants();
```

## Configuration

The constructor accepts an `apiKey` argument; application code is responsible for sourcing it. The convention is to read it from the process environment:

| Variable | Used by | Description |
|---|---|---|
| `ALTER_API_KEY` | application code (typical app workload) | Conventional name when the application owns one app-level key. |
| `AGENT_API_KEY` | application code (agent workload) | Conventional name when the workload runs as a managed agent. |

For agent workloads, the convention is to read `AGENT_API_KEY` in application code:

```typescript
import { Agent } from "@alter-ai/alter-sdk";

const agent = new Agent({ apiKey: process.env.AGENT_API_KEY! });
```

## Conventions

- Every method is `async` and returns a `Promise`. Use `await`.
- Models are frozen objects. Field names are `camelCase` (the wire format is `snake_case`; the SDK maps automatically).
- Methods that retrieve tokens never return the token plaintext. The SDK injects it into the outbound request.
- Public exceptions extend `AlterSDKError`. See [Errors](/reference/typescript-sdk/errors).
- Time-budget parameters are in **milliseconds** (matches `setTimeout`, `Date.now()`). User-selectable durations passed to the Connect UI are in **seconds**.
