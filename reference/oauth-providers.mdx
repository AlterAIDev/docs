---
title: 'OAuth providers'
description: 'Connect end-user accounts on supported OAuth providers.'
---

Alter ships with 70 pre-configured OAuth providers covering productivity, communication, CRM, finance, design, and developer tools. Each provider has a setup guide with the property table, OAuth-credential creation flow, and SDK code example.

For step-by-step setup, see [Setting up an app](/admin/setting-up-your-app#configuring-providers). For the runtime flow, see [Call APIs on behalf of users](/guides/call-apis-on-behalf-of-users).

For any provider not listed here that supports OAuth 2.0, use the **Custom OAuth** template — point Alter at the authorize / token / revoke URLs and Alter handles the rest.
