## Overview

Connect your users to Miro for collaborative whiteboards, visual project planning, and team ideation.

| Property | Value |
|----------|-------|
| **Provider ID** | `miro` |
| **Category** | Productivity |
| **PKCE** | Supported |
| **Token refresh** | Automatic |
| **Redirect URI** | Shown in [Developer Portal](https://portal.alterauth.com) |

## Step 1: Create a Miro OAuth App

<Steps>
  <Step title="Open Miro Apps settings">
    Sign in to Miro and go to [miro.com/app/settings/user-profile/apps](https://miro.com/app/settings/user-profile/apps).
  </Step>

  <Step title="Create a Developer team">
    If you don't already have one, create a Developer team — Miro requires app development to happen inside a sandboxed Developer team.
  </Step>

  <Step title="Create a new app">
    Click **+ Create new app**, enter an app name, and assign it to your Developer team.
  </Step>

  <Step title="Configure the redirect URI">
    Open the new app's settings, add the Alter callback URL from the [Developer Portal](https://portal.alterauth.com) to the **Redirect URI for OAuth2.0** field, and select the required scopes.
  </Step>

  <Step title="Get credentials">
    Copy the **Client ID** and **Client Secret**.
  </Step>
</Steps>

<Warning>
  Never share your Client Secret publicly. Store it securely and only enter it in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to the application.
  </Step>

  <Step title="Add Miro provider">
    Go to **OAuth Providers** > **Add Provider** > **Miro**.
  </Step>

  <Step title="Enter credentials">
    - **Client ID**: Paste your Miro Client ID
    - **Client Secret**: Paste your Miro Client Secret
  </Step>

  <Step title="Select scopes">
    Choose the scopes the application needs.
  </Step>

  <Step title="Save">
    Click **Save**. The provider is now active.
  </Step>
</Steps>

## Available Scopes

| Scope | Description |
|-------|-------------|
| `boards:read` | Read board content and metadata |
| `boards:write` | Create and modify boards |
| `identity:read` | Read user identity and profile |
| `team:read` | Read team information |
