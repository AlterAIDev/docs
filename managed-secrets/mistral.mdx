## Overview

Use Mistral AI credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `mistral` |
| **Category** | AI Models |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the Mistral Console">
    Log in to the Mistral Console at [console.mistral.ai](https://console.mistral.ai/).
  </Step>
  <Step title="Navigate to API Keys">
    Navigate to API Keys at [console.mistral.ai/api-keys](https://console.mistral.ai/api-keys).
  </Step>
  <Step title="Create a new key">
    Click 'Create new key'.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Mistral AI">
    Go to **Managed Secrets** > **Add Provider** > **Mistral AI**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.mistral.ai/v1/chat/completions",
)
```
