## Overview

Use Perplexity credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `perplexity` |
| **Category** | AI Models |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to Perplexity">
    Log in to Perplexity at [www.perplexity.ai](https://www.perplexity.ai/).
  </Step>
  <Step title="Go to Settings → API">
    Go to Settings → API at [www.perplexity.ai/settings/api](https://www.perplexity.ai/settings/api).
  </Step>
  <Step title="Generate a new API key">
    Generate a new API key.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Perplexity">
    Go to **Managed Secrets** > **Add Provider** > **Perplexity**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.perplexity.ai/chat/completions",
)
```
