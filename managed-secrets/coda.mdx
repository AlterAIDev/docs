## Overview

Use Coda credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `coda` |
| **Category** | Productivity |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to Coda">
    Log in to Coda at [coda.io](https://coda.io/).
  </Step>
  <Step title="Go to Account Settings">
    Go to Account Settings at [coda.io/account](https://coda.io/account).
  </Step>
  <Step title="Find the API Settings section">
    Scroll to the API Settings section.
  </Step>
  <Step title="Generate an API token">
    Click 'Generate API token'.
  </Step>
  <Step title="Copy your token">
    Copy the generated API token.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to your app.
  </Step>

  <Step title="Add Coda">
    Go to **Managed Secrets** > **Add Provider** > **Coda**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    HttpMethod.GET,
    "https://coda.io/apis/v1/docs",
    grant_id="YOUR_GRANT_ID",
)
```
