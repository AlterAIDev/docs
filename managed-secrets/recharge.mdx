## Overview

Use Recharge credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `recharge` |
| **Category** | E-Commerce |
| **Credential Type** | API Key |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the Recharge Merchant Portal">
    Log in to the Recharge Merchant Portal at [vendors.rechargepayments.com](https://vendors.rechargepayments.com/).
  </Step>
  <Step title="Go to API tokens">
    Go to API tokens in your account settings at [vendors.rechargepayments.com/tools/api](https://vendors.rechargepayments.com/tools/api).
  </Step>
  <Step title="Create an API token">
    Click 'Create an API token' and configure permissions.
  </Step>
  <Step title="Copy your token">
    Copy the generated API token.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Recharge">
    Go to **Managed Secrets** > **Add Provider** > **Recharge**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
```
