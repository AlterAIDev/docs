## Overview

Use PostHog credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `posthog` |
| **Category** | Developer Tools |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to PostHog">
    Log in to PostHog at [us.posthog.com](https://us.posthog.com/).
  </Step>
  <Step title="Go to Settings → User → Personal API Keys">
    Go to Settings → User → Personal API Keys at [us.posthog.com/settings/user-api-keys](https://us.posthog.com/settings/user-api-keys).
  </Step>
  <Step title="Create a personal API key">
    Click 'Create personal API key', set a label and permissions.
  </Step>
  <Step title="Copy your API key">
    Copy the generated Personal API Key (starts with phx_).
  </Step>
  <Step title="Enter host if needed">
    If using EU cloud or self-hosted, enter the host in the PostHog Host field.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add PostHog">
    Go to **Managed Secrets** > **Add Provider** > **PostHog**.
  </Step>

  <Step title="Enter your credentials">
    Paste your Personal API Key into the credential field. If using EU Cloud or a self-hosted instance, also enter your PostHog host (e.g. `eu.i.posthog.com`) in the PostHog Host field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.GET,
    "https://us.i.posthog.com/api/projects/YOUR_PROJECT_ID/events",  # Use eu.i.posthog.com for EU cloud or your own host for self-hosted
)
```
