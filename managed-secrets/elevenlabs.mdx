## Overview

Use ElevenLabs credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `elevenlabs` |
| **Category** | AI Models |
| **Credential Type** | API Key |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to your ElevenLabs account">
    Log in to your ElevenLabs account at [elevenlabs.io](https://elevenlabs.io/).
  </Step>
  <Step title="Go to Settings → API Keys">
    Go to Settings → API Keys at [elevenlabs.io/app/settings/api-keys](https://elevenlabs.io/app/settings/api-keys).
  </Step>
  <Step title="Create an API Key">
    Click 'Create API Key'.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add ElevenLabs">
    Go to **Managed Secrets** > **Add Provider** > **ElevenLabs**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.elevenlabs.io/v1/text-to-speech/YOUR_VOICE_ID",
)
```
