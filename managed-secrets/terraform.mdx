## Overview

Use Terraform credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `terraform` |
| **Category** | Developer Tools |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to HCP Terraform">
    Log in to HCP Terraform at [app.terraform.io](https://app.terraform.io/).
  </Step>
  <Step title="Go to User Settings → Tokens">
    Go to User Settings → Tokens at [app.terraform.io/app/settings/tokens](https://app.terraform.io/app/settings/tokens).
  </Step>
  <Step title="Create an API token">
    Click 'Create an API token'.
  </Step>
  <Step title="Copy your token">
    Copy the generated token.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Terraform">
    Go to **Managed Secrets** > **Add Provider** > **Terraform**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.GET,
    "https://app.terraform.io/api/v2/organizations",
)
```
