## Overview

Use Resend credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `resend` |
| **Category** | Communication |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the Resend Dashboard">
    Log in to the Resend Dashboard at [resend.com](https://resend.com/).
  </Step>
  <Step title="Navigate to API Keys">
    Navigate to API Keys at [resend.com/api-keys](https://resend.com/api-keys).
  </Step>
  <Step title="Create an API Key">
    Click 'Create API Key' and set permissions.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Resend">
    Go to **Managed Secrets** > **Add Provider** > **Resend**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.resend.com/emails",
)
```
