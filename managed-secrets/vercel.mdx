## Overview

Use Vercel credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `vercel` |
| **Category** | Developer Tools |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to Vercel">
    Log in to Vercel at [vercel.com](https://vercel.com/).
  </Step>
  <Step title="Go to Account Settings → Tokens">
    Go to Account Settings → Tokens at [vercel.com/account/tokens](https://vercel.com/account/tokens).
  </Step>
  <Step title="Create a token">
    Click 'Create' and set a name and scope.
  </Step>
  <Step title="Copy your token">
    Copy the generated token.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to your app.
  </Step>

  <Step title="Add Vercel">
    Go to **Managed Secrets** > **Add Provider** > **Vercel**.
  </Step>

  <Step title="Enter your credentials">
    Paste your Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_GRANT_ID",
    HttpMethod.GET,
    "https://api.vercel.com/v9/projects",
)
```
