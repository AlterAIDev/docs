## Overview

Use Axiom credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `axiom` |
| **Category** | Monitoring & Observability |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to Axiom">
    Log in to Axiom at [app.axiom.co](https://app.axiom.co/).
  </Step>
  <Step title="Go to Settings → API Tokens">
    Go to Settings → API Tokens at [app.axiom.co/settings/api-tokens](https://app.axiom.co/settings/api-tokens).
  </Step>
  <Step title="Create a new token">
    Click 'New API Token' and configure permissions.
  </Step>
  <Step title="Copy your token">
    Copy the generated API token.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to your app.
  </Step>

  <Step title="Add Axiom">
    Go to **Managed Secrets** > **Add Provider** > **Axiom**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    HttpMethod.POST,
    "https://api.axiom.co/v1/datasets/YOUR_DATASET/ingest",
    grant_id="YOUR_GRANT_ID",
)
```
