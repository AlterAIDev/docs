## Overview

Use Rootly credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `rootly` |
| **Category** | Monitoring & Observability |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to Rootly">
    Log in to Rootly at [rootly.com](https://rootly.com/).
  </Step>
  <Step title="Go to Account → API Keys">
    Go to Account → API Keys at [rootly.com/account/api-keys](https://rootly.com/account/api-keys).
  </Step>
  <Step title="Generate a new key">
    Click 'Generate New Key'.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to your app.
  </Step>

  <Step title="Add Rootly">
    Go to **Managed Secrets** > **Add Provider** > **Rootly**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_GRANT_ID",
    HttpMethod.GET,
    "https://api.rootly.com/v1/incidents",
)
```
