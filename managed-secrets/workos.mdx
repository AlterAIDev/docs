## Overview

Use WorkOS credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `workos` |
| **Category** | Developer Tools |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the WorkOS Dashboard">
    Log in to the WorkOS Dashboard at [dashboard.workos.com](https://dashboard.workos.com/).
  </Step>
  <Step title="Go to API Keys">
    Go to API Keys at [dashboard.workos.com/api-keys](https://dashboard.workos.com/api-keys).
  </Step>
  <Step title="Copy your API Key">
    Copy your API Key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to your app.
  </Step>

  <Step title="Add WorkOS">
    Go to **Managed Secrets** > **Add Provider** > **WorkOS**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    HttpMethod.GET,
    "https://api.workos.com/sso/connections",
    grant_id="YOUR_GRANT_ID",
)
```
