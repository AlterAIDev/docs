## Overview

Use OpenAI credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `openai` |
| **Category** | AI Models |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the OpenAI Platform">
    Log in to the OpenAI Platform at [platform.openai.com](https://platform.openai.com/).
  </Step>
  <Step title="Navigate to API Keys">
    Navigate to API Keys at [platform.openai.com/api-keys](https://platform.openai.com/api-keys).
  </Step>
  <Step title="Create a new secret key">
    Click 'Create new secret key' and give it a name.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key (it won't be shown again).
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add OpenAI">
    Go to **Managed Secrets** > **Add Provider** > **OpenAI**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.openai.com/v1/chat/completions",
)
```
