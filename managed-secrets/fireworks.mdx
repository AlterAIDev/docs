## Overview

Use Fireworks AI credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `fireworks` |
| **Category** | AI Models |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to Fireworks AI">
    Log in to Fireworks AI at [fireworks.ai](https://fireworks.ai/).
  </Step>
  <Step title="Go to Account → API Keys">
    Go to Account → API Keys at [fireworks.ai/account/api-keys](https://fireworks.ai/account/api-keys).
  </Step>
  <Step title="Create an API Key">
    Click 'Create API Key'.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Fireworks AI">
    Go to **Managed Secrets** > **Add Provider** > **Fireworks AI**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.fireworks.ai/inference/v1/chat/completions",
)
```
