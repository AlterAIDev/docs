## Overview

Use Anthropic credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `anthropic` |
| **Category** | AI Models |
| **Credential Type** | API Key |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the Anthropic Console">
    Log in to the Anthropic Console at [console.anthropic.com](https://console.anthropic.com/).
  </Step>
  <Step title="Navigate to Settings → API Keys">
    Navigate to Settings → API Keys.
  </Step>
  <Step title="Create a new key">
    Click 'Create Key' and give it a name.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Anthropic">
    Go to **Managed Secrets** > **Add Provider** > **Anthropic**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.anthropic.com/v1/messages",
)
```
