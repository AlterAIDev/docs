## Overview

Use fal.ai credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `falai` |
| **Category** | AI Models |
| **Credential Type** | API Key |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the fal.ai Dashboard">
    Log in to the fal.ai Dashboard at [fal.ai/dashboard](https://fal.ai/dashboard).
  </Step>
  <Step title="Navigate to Keys">
    Navigate to Keys in the dashboard at [fal.ai/dashboard/keys](https://fal.ai/dashboard/keys).
  </Step>
  <Step title="Create a key">
    Click 'Create Key'.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add fal.ai">
    Go to **Managed Secrets** > **Add Provider** > **fal.ai**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://fal.run/fal-ai/flux/dev",

```
