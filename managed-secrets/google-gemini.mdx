## Overview

Use Google Gemini credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `google-gemini` |
| **Category** | AI Models |
| **Credential Type** | API Key |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Sign in to Google">
    Sign in to your Google account at [accounts.google.com](https://accounts.google.com/).
  </Step>
  <Step title="Go to Google AI Studio API Keys">
    Navigate to [Google AI Studio API Keys](https://aistudio.google.com/apikey).
  </Step>
  <Step title="Create API key">
    Click 'Create API key' and select a Google Cloud project.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Google Gemini">
    Go to **Managed Secrets** > **Add Provider** > **Google Gemini**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://generativelanguage.googleapis.com/v1/models/gemini-1.5-flash:generateContent",
)
```
