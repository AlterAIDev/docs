## Overview

Use SendGrid credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `sendgrid` |
| **Category** | Communication |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to the SendGrid Dashboard">
    Log in to the SendGrid Dashboard at [app.sendgrid.com](https://app.sendgrid.com/).
  </Step>
  <Step title="Go to Settings → API Keys">
    Go to Settings → API Keys at [app.sendgrid.com/settings/api_keys](https://app.sendgrid.com/settings/api_keys).
  </Step>
  <Step title="Create an API Key">
    Click 'Create API Key' and set permissions.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key (it won't be shown again).
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to your app.
  </Step>

  <Step title="Add SendGrid">
    Go to **Managed Secrets** > **Add Provider** > **SendGrid**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_GRANT_ID",
    HttpMethod.POST,
    "https://api.sendgrid.com/v3/mail/send",
)
```
