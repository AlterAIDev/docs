## Overview

Use Replicate credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `replicate` |
| **Category** | AI Models |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to Replicate">
    Log in to Replicate at [replicate.com](https://replicate.com/).
  </Step>
  <Step title="Go to Account → API Tokens">
    Go to Account → API Tokens at [replicate.com/account/api-tokens](https://replicate.com/account/api-tokens).
  </Step>
  <Step title="Create a token">
    Click 'Create Token' and give it a name.
  </Step>
  <Step title="Copy your token">
    Copy the generated API token.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterai.dev](https://portal.alterai.dev) and navigate to your app.
  </Step>

  <Step title="Add Replicate">
    Go to **Managed Secrets** > **Add Provider** > **Replicate**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Token into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `connection_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    "YOUR_CONNECTION_ID",
    HttpMethod.POST,
    "https://api.replicate.com/v1/predictions",
)
```
