## Overview

Use PagerDuty credentials to make authenticated API calls through Alter Vault without exposing your API keys in code.

| Property | Value |
|----------|-------|
| **Provider ID** | `pagerduty` |
| **Category** | Monitoring & Observability |
| **Credential Type** | Bearer Token |

## Step 1: Get Your Credentials

<Steps>
  <Step title="Log in to PagerDuty">
    Log in to PagerDuty at [app.pagerduty.com](https://app.pagerduty.com/).
  </Step>
  <Step title="Go to Integrations → Developer Tools → API Access Keys">
    Go to Integrations → Developer Tools → API Access Keys at [app.pagerduty.com/api_access_keys](https://app.pagerduty.com/api_access_keys).
  </Step>
  <Step title="Create a new API Key">
    Click 'Create New API Key'.
  </Step>
  <Step title="Copy your API key">
    Copy the generated API key.
  </Step>
</Steps>

<Warning>
  Never share your credentials publicly. Store them securely and only enter them in the Alter Vault Developer Portal.
</Warning>

## Step 2: Add to Alter Vault

<Steps>
  <Step title="Open the Developer Portal">
    Go to [portal.alterauth.com](https://portal.alterauth.com) and navigate to your app.
  </Step>

  <Step title="Add PagerDuty">
    Go to **Managed Secrets** > **Add Provider** > **PagerDuty**.
  </Step>

  <Step title="Enter your credentials">
    Paste your API Key into the credential field.
  </Step>

  <Step title="Save">
    Click **Save**. You'll receive a `grant_id` to use with the SDK.
  </Step>
</Steps>

## Using in Code

```python
response = await vault.request(
    HttpMethod.GET,
    "https://api.pagerduty.com/incidents",
    grant_id="YOUR_GRANT_ID",
)
```
